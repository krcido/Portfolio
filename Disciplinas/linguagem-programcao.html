O que é Linguagem de Programação?

Uma breve conceituação

A linguagem de programação é uma forma de comunicação entre humanos e computadores, utilizada para criar programas e instruções que o computador pode entender e executar. É uma ferramenta fundamental para desenvolver software, aplicativos e sistemas, permitindo a criação de soluções complexas para uma variedade de problemas.

Linguagens de programação são conjuntos de regras e instruções utilizadas para escrever programas de computador. Elas permitem que os desenvolvedores comuniquem suas intenções e comandos de maneira estruturada, facilitando a criação de software.

Existem diversas linguagens de programação disponíveis, cada uma com suas características e finalidades específicas. Algumas são voltadas para a criação de aplicativos web, como JavaScript e Python. Outras são mais adequadas para a programação de sistemas operacionais, como C e C++. Há também linguagens de programação específicas para o desenvolvimento de aplicativos móveis, como Swift (iOS) e Java (Android).

Cada linguagem de programação possui sua própria sintaxe, ou seja, a forma correta de escrever o código. Além disso, elas podem ter diferentes paradigmas de programação, como a programação orientada a objetos, a programação funcional ou a programação procedural.

Existem diversos tipos de linguagens de programação, cada uma com suas características e finalidades específicas.

Linguagens de baixo nível
São mais próximas da linguagem de máquina e fornecem maior controle sobre o hardware do computador. Exemplos incluem Assembly e C.
Linguagens de alto nível
Projetadas para serem mais próximas da linguagem humana, facilitando a escrita e a compreensão do código. Exemplos: Python, Java, C++ ...
Linguagens de Script
Projetadas para execução de scripts em ambientes específicos, como servidores web ou sistemas operacionais. Exemplos JavaScript, Python ...
Linguagens Orientadas a Objetos - POO
Os programas são estruturados em torno de objetos que possuem atributos e comportamentos. Exemplos incluem Java, C++, Python e C#.
Linguagens de manipulação de dados
São usadas principalmente para manipular e consultar bancos de dados.Exemplos incluem SQL e LINQ.

DOM

O DOM (Document Object Model) é uma interface de programação que representa a estrutura de um documento HTML ou XML como uma árvore de objetos. Ele permite que os programadores acessem, manipulem e atualizem dinamicamente os elementos e conteúdos de uma página web.

Nesse projeto, há uma interação com o DOM para adicionar comportamentos a um elemento específico identificado pelo ID "click". Quando o cursor do mouse entra na área do elemento, a função "entrar" é acionada, alterando a cor de fundo para azul. Quando o cursor do mouse sai da área do elemento, a função "sair" é acionada, alterando a cor de fundo para verde e atualizando o texto do elemento para "Clique em mim!". Quando o elemento é clicado, a função "clicar" é acionada, alterando a cor de fundo para vermelho e atualizando o texto do elemento para "Clicou!".

Portanto, esse projeto de software demonstra o uso do DOM para adicionar interatividade a um elemento específico em uma página web. Ele ilustra como as propriedades e métodos do DOM podem ser utilizados para manipular dinamicamente os elementos da página em resposta a eventos do usuário, como passar o mouse sobre o elemento e clicar nele.


Aqui estão alguns detalhes sobre o software:


  • 1 - Acessando elementos do DOM:

    window.document.getElementById("click"): Essa linha de código utiliza o método getElementById do objeto document para selecionar um elemento no DOM com o ID "click" e armazená-lo na variável box. Isso permite acessar e manipular as propriedades desse elemento.

  • 2 - Adicionando Event Listeners:

    box.addEventListener("mouseenter", entrar): Esse código adiciona um ouvinte de evento para o evento "mouseenter" (quando o cursor do mouse entra na área do elemento) ao elemento box. Quando esse evento ocorrer, a função entrar será chamada. box.addEventListener

    ("mouseout", sair): Essa linha adiciona um ouvinte de evento para o evento "mouseout" (quando o cursor do mouse sai da área do elemento). Quando esse evento ocorrer, a função sair será chamada.

    box.addEventListener("click", clicar): Essa linha adiciona um ouvinte de evento para o evento "click" (quando o elemento é clicado). Quando esse evento ocorrer, a função clicar será chamada.
  • Manipulação de propriedades do elemento:

    box.style.background = "blue": Essa linha altera a propriedade background do elemento box para a cor azul, alterando assim o fundo do elemento.br

    box.innerText = "Clique em mim!": Essa linha altera o conteúdo de texto do elemento box, substituindo-o por "Clique em mim!".

    box.style.background = "green": Essa linha altera a propriedade background do elemento box para a cor verde, alterando novamente o fundo do elemento.

    box.innerText = "Clicou!": Essa linha altera o conteúdo de texto do elemento box, substituindo-o por "Clicou!".

    box.style.background = "red": Essa linha altera a propriedade background do elemento box para a cor

Essas são algumas das principais técnicas utilizadas no código para manipular o DOM. Por meio do acesso aos elementos do DOM e da manipulação de suas propriedades, é possível criar interações dinâmicas e personalizadas em uma página web.

Inteli Code: Um Gerador de Páginas Interativas


O Inteli Code foi um projeto inovador e desafiador. Ele permite aos usuários criar páginas da web interativas de forma rápida e fácil. Com essa ferramenta, os usuários podem inserir seu próprio código HTML e JavaScript em caixas de texto separadas e o software irá combinar esses códigos, aplicar estilos pré-definidos e executar a lógica fornecida, gerando uma página HTML estilizada e funcional em tempo real.


A interface do Inteli Code é intuitiva e amigável. Os usuários são apresentados com duas áreas de texto distintas. Na primeira caixa de texto, eles podem inserir seu código HTML, definindo a estrutura e os elementos da página. Na segunda caixa de texto, é possível adicionar o código JavaScript para fornecer a lógica e a interatividade à página.


Após inserir o código nos respectivos campos, basta clicar no botão "Gerar Página" e o Inteli Code entra em ação. O software combina os códigos HTML e JavaScript fornecidos e utiliza técnicas de manipulação do DOM para criar dinamicamente a página dentro da própria interface.


Uma vez gerada a página, o Inteli Code aplica estilos pré-definidos para tornar a página visualmente atraente. Os elementos HTML são formatados e posicionados de maneira agradável, tornando a página estilizada e agradável aos olhos.


Além disso, o código JavaScript fornecido é executado, permitindo que a lógica seja implementada e a interatividade seja adicionada à página gerada. Isso significa que os usuários podem criar elementos interativos, como botões, formulários, animações e muito mais.


Nesse projeto aprendemos algumas coisas diferentes em HTML como:

HTML Semântico

HTML Semântico é uma abordagem na construção de páginas da web que visa atribuir significado estrutural e contextual aos elementos HTML. Em vez de usar tags apenas para fins de formatação visual, o HTML Semântico busca dar sentido ao conteúdo, permitindo uma compreensão mais clara tanto pelos humanos quanto pelos mecanismos de busca.


Ao utilizar o HTML Semântico, os desenvolvedores se concentram em escolher as tags HTML apropriadas com base no propósito e na função dos elementos na página. Em vez de usar <div>s genéricos para agrupar o conteúdo, são utilizadas tags mais descritivas, como <header>, <nav>, <main>, <article>, <section>, <footer>, entre outras.


Essas tags específicas fornecem uma estrutura clara e significativa para o conteúdo da página. Por exemplo, o <header> é usado para o cabeçalho da página, o <nav> para a navegação, o <main> para o conteúdo principal, o <article> para conteúdo independente e o <footer> para o rodapé.


Ao aplicar corretamente o HTML Semântico, além de tornar o código mais legível e compreensível para os desenvolvedores, também traz benefícios para a acessibilidade e a indexação pelos mecanismos de busca. Os leitores de tela podem interpretar melhor a estrutura da página e os mecanismos de busca podem entender o contexto e o propósito dos elementos, o que pode melhorar a classificação nos resultados de pesquisa.


Além disso, o HTML Semântico também facilita a estilização e a manutenção do código, pois os seletores CSS podem ser direcionados a elementos semânticos específicos, simplificando a aplicação de estilos e a compreensão da estrutura da página.


Em resumo, o HTML Semântico é uma abordagem poderosa para a construção de páginas da web, priorizando o significado e a estrutura do conteúdo. Ao utilizar tags HTML descritivas e contextuais, os desenvolvedores podem criar páginas mais acessíveis, compreensíveis pelos mecanismos de busca e mais fáceis de estilizar e manter.


Concatenação de elementos HTML

A concatenação de objetos HTML usando o DOM (Document Object Model) é uma técnica poderosa para criar estruturas complexas de elementos HTML de forma dinâmica. Ao manipular o DOM, é possível criar, modificar e combinar elementos HTML em tempo real, permitindo a construção flexível e personalizada de páginas da web.


Depois de criar os elementos, podemos utilizar métodos em JavaScript para adicionar esses elementos como filhos de outros elementos. Isso permite a criação de estruturas hierárquicas, onde um elemento HTML é aninhado dentro de outro.


Aqui está um exemplo de código:



															const divContainer = document.createElement('div');

															const paragraph1 = document.createElement('p');
															paragraph1.textContent = 'Este é o primeiro parágrafo.';
															divContainer.appendChild(paragraph1);

															const paragraph2 = document.createElement('p');
															paragraph2.textContent = 'Este é o segundo parágrafo.';
															divContainer.appendChild(paragraph2);

Dessa forma, podemos criar e combinar elementos HTML dinamicamente, criando uma estrutura complexa de conteúdo. Podemos repetir esse processo para adicionar mais elementos e aninhar elementos dentro de outros elementos, dependendo das necessidades do projeto.


A concatenação de objetos HTML usando o DOM oferece flexibilidade e controle total sobre a criação e manipulação dos elementos HTML. Isso é especialmente útil ao lidar com conteúdo dinâmico, como exibição de dados de um banco de dados ou a criação de componentes reutilizáveis.


Hover - CSS

O efeito hover é uma funcionalidade amplamente utilizada no CSS para criar interatividade em elementos HTML. Quando um elemento recebe o hover, isso significa que o cursor do mouse está sobre ele. O efeito hover permite que você defina estilos diferentes para um elemento quando o cursor está sobre ele, fornecendo feedback visual ao usuário.


Ao aplicar o efeito hover a um elemento, podemos especificar regras de estilo que serão ativadas apenas quando o elemento estiver sendo hover. Essas regras podem alterar a cor de fundo, a cor do texto, a opacidade, o tamanho, a posição ou qualquer outra propriedade que você deseje modificar.


O efeito hover é especialmente útil para melhorar a usabilidade de um site, destacando elementos interativos, como links, botões ou menus. Quando o usuário passa o cursor sobre esses elementos, eles podem mudar de cor, aumentar de tamanho ou exibir animações sutis, proporcionando uma resposta visual clara de que o elemento está interativo.


Aqui está um exemplo de código:

															
																a:hover {
																color: red;
																}

Dessa forma, quando o usuário passar o cursor sobre um link ``, o texto mudará para a cor vermelha. Essa mudança de estilo ocorrerá apenas durante o hover e retornará ao estilo original assim que o cursor for movido para fora do elemento.


O efeito hover também pode ser aplicado a outros tipos de elementos, como botões, imagens, divs, entre outros. Combinado com transições ou animações, o efeito hover pode adicionar uma interatividade sutil, tornando a experiência do usuário mais envolvente.


É importante lembrar de equilibrar o uso do efeito hover para evitar uma sobrecarga visual excessiva. Aplique-o de forma consistente e intuitiva para criar uma experiência de usuário agradável e intuitiva.

Calculadora de Folha de Pagamento

Este projeto consiste em um sistema de cálculo de folha de pagamento, desenvolvido utilizando a linguagem de programação JavaScript em conjunto com HTML e CSS. O objetivo é permitir que o usuário insira o salário bruto e o número de dependentes, e então o sistema realiza os cálculos necessários para determinar os descontos de INSS, IRRF e vale-transporte, além de calcular o salário líquido.

O código possui três principais funções:

  1. 1 - A função calcular(): Esta função é executada quando o botão "Calcular" é clicado. Ela recupera os valores inseridos pelo usuário, realiza os cálculos e exibe os resultados na página HTML.

  2. 2 - A função resetar(): Esta função é executada quando o botão "Resetar" é clicado. Ela permite que o usuário altere os valores de salário bruto e número de dependentes novamente, tornando os campos editáveis.

  3. 3 - A função ocultar(): Esta função é executada quando o botão "Resetar" é clicado. Ela oculta a seção de resultados da folha de pagamento, para que os valores não fiquem visíveis após o reset.

A estrutura básica do código utiliza o método getElementById para selecionar elementos HTML específicos, como botões e campos de entrada. Os eventos addEventListener são utilizados para atribuir funções aos botões, de modo que quando são clicados, as funções correspondentes são executadas.


Os valores dos campos de entrada são recuperados utilizando o getElementById e armazenados em variáveis. Em seguida, são realizados os cálculos de INSS, base de cálculo do IRRF, IRRF e vale-transporte, utilizando condicionais (if, else if e else). Os resultados são exibidos na página HTML utilizando o método innerHTML.


Após o cálculo dos descontos e do salário líquido, a função display é utilizada para exibir a seção de resultados da folha de pagamento (document.getElementById("saida").style.display = "flex"). Além disso, os campos de entrada são travados para evitar a edição após o cálculo (document.getElementById("salarioBruto").readOnly = true; e document.getElementById("dependentes").readOnly = true;).


O projeto utiliza HTML e CSS para estruturar e estilizar a página, enquanto o JavaScript é responsável pela lógica de cálculo e interação com o usuário.


Outros tópicos interessantes desenvolvidos no projeto foram:


Uma função muito interessante que foi desenvolvida nesse trabalho é a CSS display. Foi trabalhado com o formato "none".


A técnica display: none; é usada em CSS para ocultar elementos HTML em uma página. Quando aplicada a um elemento, essa propriedade faz com que o elemento não seja exibido na página, ocupando nenhum espaço visual.


Ao definir display: none; para um elemento, ele desaparece completamente da página, como se não estivesse presente. Isso significa que ele não será renderizado e nenhum espaço será reservado para ele na disposição dos outros elementos.


Essa técnica é útil quando você deseja ocultar temporariamente um elemento ou controlar sua visibilidade dinamicamente por meio de JavaScript. Por exemplo, você pode ocultar um formulário até que um botão seja clicado e, em seguida, exibi-lo alterando a propriedade display para block ou inline, conforme necessário.


Ao modificar o valor de display, você pode controlar como o elemento é exibido. Alguns valores comuns são:


  • block: o elemento é exibido como um bloco, ocupando todo o espaço horizontal disponível. Ele inicia uma nova linha antes e depois do elemento.
  • inline: o elemento é exibido como uma caixa inline, ocupando apenas o espaço necessário para seu conteúdo. Ele não inicia uma nova linha.
  • inline-block: o elemento é exibido como uma caixa inline, mas permite definir largura, altura, margens e preenchimento. Ele não inicia uma nova linha.
  • none: o elemento é completamente oculto e não afeta o layout da página.

É importante notar que, ao aplicar display: none; a um elemento, ele não será visível para os usuários, mas ainda existirá no código HTML. Isso significa que você pode alternar sua visibilidade novamente definindo display para um valor diferente, se necessário.

Aprendendo Programação Orientada a Objetos

Esse projeto consistiu em aplicar POO para a construção de um sotfware de gerenciamento de estoque. Ele foi feito em um curso a parte da faculde, porém, ministrado no próprio Senai.


Ele utiliza conceitos de estruturas de classes, sintaxe Java, técnicas de programação procedural e interação com o usuário por meio da entrada e saída do console.
O programa está dividido em duas classes: "Produto" e "produtoTester". Vamos analisar cada uma delas.


1 - Classe "Produto":


  • idProduto (int): número identificador do produto.
  • nome (String): nome do produto.
  • preco (double): preço do produto.
  • quantidade (int): quantidade em estoque do produto.
  • estado (boolean): indica se o produto está ativo (true) ou descontinuado (false).

A classe possui três construtores:

  • O primeiro é um construtor vazio (default).
  • O segundo recebe os parâmetros idProduto e nome para criar um novo objeto Produto com valores iniciais para esses atributos.
  • O terceiro construtor recebe todos os atributos do produto e os define nos campos correspondentes.

A classe também possui métodos getters e setters para cada atributo, permitindo acessar e modificar seus valores.
Além disso, a classe possui o método valorEstoque(), que calcula o valor total do estoque multiplicando o preço pelo quantidade.
O método toString() é sobrescrito para retornar uma representação em forma de String do produto, incluindo todos os atributos.

2 - Calasse Product Tester (Main):

A classe é responsável por executar o programa principal e interagir com o usuário através do console.


Ela contém uma lista de produtos (listaProdutos) e um objeto Scanner (scanner) para ler a entrada do usuário.

O método main() é o ponto de entrada do programa. Ele inicializa a lista de produtos com alguns produtos iniciais e entra em um loop onde exibe um menu de opções para o usuário.

O método menu() exibe o menu de opções para o usuário e lê a opção escolhida. Ele retorna a opção selecionada.

O método clearCache() é usado para limpar o cache do scanner após a leitura de um número inteiro.

O método cadastrar() solicita ao usuário as informações necessárias para cadastrar um novo produto e adiciona o produto à lista de produtos.

O método adicionar() solicita ao usuário o número do produto e a quantidade a ser adicionada ao estoque. Em seguida, ele busca o produto correspondente na lista e atualiza a quantidade.

O método remover() funciona de maneira semelhante ao método adicionar(), mas remove a quantidade especificada do estoque.

O método modificar() permite ao usuário selecionar um produto pelo número e, em seguida, escolher qual atributo deseja modificar (nome, preço, quantidade ou estado). O método menuAlterar() exibe o menu de opções para a modificação e retorna a opção selecionada.

O método listar() exibe na tela a lista de produtos com seus respectivos atributos.

Java - Projeto Screenshot

Aprendendo Programação Orientada a Objetos

Inicialmente, criamos uma classe abstrata chamada "Shape", que representa a abstração de uma figura geométrica que será exibida no papel de parede. Essa classe possui métodos abstratos para desenhar as figuras.

Em seguida, implementamos subclasses concretas para diferentes tipos de figuras, como "Círculo" e "Retângulo". Cada uma dessas subclasses herda da classe "Shape" e implementa seus métodos de acordo com as características específicas de cada figura.

Através do polimorfismo, podemos criar um array de objetos do tipo "Shape" e armazenar diferentes tipos de figuras nesse array. Isso nos permite tratar todas as figuras de forma genérica, independentemente do tipo específico de cada uma delas.

Ao exibir o papel de parede, percorremos o array de figuras e chamamos o método de desenho para cada uma delas. O polimorfismo garante que o método adequado de cada figura seja invocado, independentemente do seu tipo concreto.

Além disso, utilizamos tratamento de colisões para evitar que as figuras se sobreponham umas às outras. Cada figura possui um método para verificar se houve colisão com outras figuras no papel de parede. Caso ocorra uma colisão, podemos implementar a lógica apropriada, como mudar a cor da figura ou reposicioná-la.

Outro conceito visto no exercício foi a classe Swing.

A classe `javax.swing` é um pacote no Java que fornece componentes e funcionalidades para criar interfaces gráficas de usuário (GUI). Ela contém várias classes e subpacotes que permitem a criação de janelas, botões, caixas de diálogo, barras de rolagem e outros elementos de interface. Além disso, a classe `javax.swing` oferece recursos avançados, como layout de componentes, eventos de interação do usuário e renderização personalizada.

Algumas classes importantes do pacote `javax.swing` incluem:

  • - `JFrame`: Uma classe que representa uma janela ou quadro de aplicativo.

  • - `JPanel`: Uma classe que fornece um contêiner para agrupar e organizar componentes.

  • - `JButton`: Uma classe que cria um botão clicável.

  • - `JTextField`: Uma classe que cria uma caixa de texto para entrada do usuário.

  • - `JLabel`: Uma classe que cria um rótulo para exibir texto ou imagem.

  • - `JComboBox`: Uma classe que cria uma lista suspensa para seleção de opções.

  • - `JCheckBox`: Uma classe que cria uma caixa de seleção.

  • - `JRadioButton`: Uma classe que cria um botão de opção.

  • - `JScrollPane`: Uma classe que adiciona barras de rolagem a componentes que possuem conteúdo maior do que a área visível.
